# proto-ios


## **Настройка проекта:**

1. Открыть папку с проектом
2. Открыть терминал
3. pod install
4. Открыть проект
5. Clean build
6. Запустить проект

##  **Настройка сниппетов модулей: **

**1. Открыть root папку**

**2. Скопировать файл**

> Default Module.xctemplate

**3. Скопированный файл перенести в**

> ~/Library/Developer/Xcode/Templates
